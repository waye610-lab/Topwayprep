Topway Prep — SHSAT EXAM-1 GitHub Pages Upload Guide

Files included:
- index.html
- exam-text-data.js
- exam-img/ folder
- .nojekyll

How to upload to GitHub Pages:
1. Create a new GitHub repository, or open an existing repository.
2. Upload all files from this folder to the repository root. Make sure index.html is at the top level.
3. Go to Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select main branch and /root folder, then Save.
6. Open the GitHub Pages URL after deployment finishes.

Notes:
- This is a static HTML/CSS/JavaScript exam page. No backend server is required.
- Students' answers and scores are saved only in the browser on that device via localStorage.
- The loaded answer key contains 100 answers for automatic scoring.
